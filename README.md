# 🌟 がんばったねポイントシステム (Good Job Point System)

「がんばったねポイントシステム」は、子どもが「おてつだい」や「おべんきょう」を通じて楽しくポイントを貯め、景品と交換できるWebアプリケーションです。QRコードを用いて子どもを識別し、ポイントの付与や利用には親のパスコードまたは生体認証（顔認証・指紋認証など）での承認を必須とする、安心かつ親しみやすい設計になっています。

## ✨ 主な機能

* **📱 QRコードによる簡単ログイン**: 子どもごとの専用QRコードをカメラで読み取るだけで、自分のポイント画面にアクセスできます。
* **🛡️ 親の安心承認システム**: ポイントの獲得や利用の際には、親の承認（PINコードまたはWebAuthnによる生体認証/パスキー）が必要です。
* **🎁 楽しいごほうび交換**: 貯めたポイントは、事前に設定した景品と交換可能。達成時には華やかなアニメーションで盛り上げます。
* **🍱 ふりがな対応**: すべての項目名（名前、タスク、カテゴリ等）に `[漢字(ふりがな)]` 形式でふりがなを振ることができます。
* **🎨 豊かなカスタマイズ**: 子どもごとにアバター画像やヘッダー画像を設定でき、自分専用の特別なページを作れます。
* **🛠️ 充実した管理画面**: 親向けの管理画面から、子どもの追加、タスク・景品の編集、ポイントの直接調整、詳細な履歴確認が可能です。
* **🔄 複数デバイス同期**: PHPとJSONを用いたシンプルなバックエンドにより、家族の複数のスマホやPC間でデータを同期できます。

## 🚀 使い方 (ローカル環境での実行)

### 前提条件
* Node.js (v18以降を推奨)
* npm または yarn

### インストールと起動
```bash
# リポジトリのクローン
git clone https://github.com/qlonix/good-job-point-system.git
cd good-job-point-system

# 依存関係のインストール
npm install

# 開発サーバーの起動 (フロントエンドとバックエンドの両方が起動します)
npm run dev
```

起動後、ブラウザで **[http://localhost:5173](http://localhost:5173)** にアクセスしてください。管理画面の初期PINコードは `0000` です。

## 🛠 技術スタック
* **Frontend**: React 19, Vite, React Router
* **Backend**: Node.js, Express, SQLite (better-sqlite3)
* **Styling**: Vanilla CSS (パステルカラーで親しみやすいプレミアムデザイン)
* **QR Reader**: html5-qrcode, qrcode.react
* **Auth**: WebAuthn API (デバイスの生体認証・パスキー)
